# flavorcss

Runtime Build Atomic Class

## Install

Use unpkg:

```html
<script src="https://unpkg.com/flavorcss@0.4.0/umd/index.js"></script>
```

Use npm:

```sh
npm i flavorcss
```

Import in your project

```js
import "flavorcss";
```
