# flavorcss

Runtime Build Atomic Class

## Document: [https://flavor.writeflowy.com](https://flavor.writeflowy.com)

## Install

Use unpkg:

```html
<script
  type="module"
  src="https://unpkg.com/flavorcss@0.5.3/es/index.js"
></script>
```

Use npm:

```sh
npm i flavorcss
```

Import in your project

```js
import "flavorcss";
```
