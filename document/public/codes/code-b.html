Rule 1: Use values

An runtime library with parameters; it's easy to customize hundreds of atomic classes; it's easy to implement media query and pseudo class, and it's perfect with CSS variables.

Try change right code, you can use any values:

font-size:40px -> font-size:50px

And you can use css variables:

font-size:50px -> font-size:var(--title)

Use sort css variables, ignore var(...):

font-size:var(--title) -> font-size:--title
